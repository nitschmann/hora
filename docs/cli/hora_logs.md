## hora logs

Display background (daemon) tracker logs

### Synopsis

Display logs from the background (daemon) tracker. Use --follow to tail the logs in real-time.

```
hora logs [flags]
```

### Options

```
  -f, --follow   Follow log output (like tail -f)
  -h, --help     help for logs
```

```

```
