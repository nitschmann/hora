## hora ui

Start the web UI

### Synopsis

Start the web UI for time tracking with interactive charts and analytics.

```
hora ui [flags]
```

### Options

```
  -h, --help       help for ui
  -p, --port int   Port to run the web UI on (default 8080)
```
